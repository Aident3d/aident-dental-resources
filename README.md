## 📋 About This Repository

This repo provides free resources for digital dentistry professionals using Aident products:
- STL model examples
- Scanning & printing workflows
- Recommended parameters
- Python scripts for data processing

Helping you achieve a fast **Scan → Design → Print** workflow.

## 📁 Repository Structure

- `/models/` — Ready-to-print STL files (crowns, bridges, implant guides)
- `/parameters/` — Optimized slicing settings for Aident printers
- `/scripts/` — Python tools for scan data processing
- `/docs/` — Detailed English tutorials and best practices

## 🚀 Quick Start Guide

1. Clone this repository
2. Download Aident software from [Download Center](https://www.aident3d.com/download)
3. Scan with AI-30 → Design in Exocad/3Shape → Print with Aident 3D Printer
